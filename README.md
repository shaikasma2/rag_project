# RAG Project

## Overview
This project is a simple Retrieval-Augmented Generation (RAG) system.

It answers questions based on PDF documents using embeddings and vector search.

## Project Structure
- data/ → contains PDF files
- main.py → main program
- README.md → project description

## How to Run
1. Install libraries
2. Run:
   python main.py

## Features
- Loads PDF documents
- Splits text into chunks
- Converts to embeddings
- Stores in vector database
- Answers user questions
